How to run from VSCODE

step 1. pip install virtualenv from codejana_flask (the outermost version)
step 2. virtualenv venv
step 3. venv\Scripts\activate


step 1. You must be in this  directory: 314App/
step 2. pip install flask, pip install SQLAlchemy
step 3. to run website, type in the terminal: python .\run.py
step 3. click link
step 4. Ctrl+C -> deactivate

NOTE: if you run 'python .\run.py' and it tells you that certain module names are unknown, pip install 'modulename' in the terminal
until you can run the code successfully, but for most computers, pip installing flask and SQLAlchemy should be enough
